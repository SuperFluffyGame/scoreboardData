ScoreboardData
==============
A simple gametest library for saving persistant data. It uses scoreboards to store and retrieve any type of data, from integers, floats, and strings


How to Use
----------


**Set Functions**
- setInt(key, value)
- setFloat(key, value)
- setString(key, value)

**Get Functions**
- getInt(key)
- getFloat(key)
- getString(key)
